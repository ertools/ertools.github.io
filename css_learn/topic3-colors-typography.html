Colors & Typography

Color and type are the two most visible design decisions on any web page. CSS gives you powerful, flexible tools for both — from modern color formats like HSL and OKLCH to fluid typography with clamp(). Get these right, and your pages will look polished and professional.

CSS — Color Formats
.element { /* Hex (3 or 6 digits) */ color: #ff6b6b; /* opaque */ color: #ff6b6b80; /* hex with alpha channel */ /* RGB / RGBA */ background: rgb(102 126 234); /* modern syntax */ background: rgb(102 126 234 / 50%); /* with alpha */ /* HSL — most intuitive for designers */ color: hsl(232 76% 66%); /* hue saturation lightness */ color: hsl(232 76% 66% / 0.8); /* with alpha */ /* Named colors */ border-color: currentColor; /* inherits text color */ background: transparent; }
Why HSL is Great for Design

HSL (Hue, Saturation, Lightness) makes it easy to create color variations. Want a darker shade? Lower the lightness. Want a muted tone? Lower the saturation. hsl(232 76% 66%) → lighter: hsl(232 76% 80%) → darker: hsl(232 76% 40%).

Gradients

CSS — Gradients
/* Linear gradient — direction, then color stops */ .hero { background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); } /* Radial gradient */ .spotlight { background: radial-gradient(circle at center, #fbbf24 0%, #ef4444 100%); } /* Multi-stop gradient */ .rainbow { background: linear-gradient(90deg, #f43f5e, #fb923c, #facc15, #4ade80, #60a5fa, #a78bfa); } /* Gradient text (with clip) */ .gradient-text { background: linear-gradient(135deg, #667eea, #764ba2); -webkit-background-clip: text; -webkit-text-fill-color: transparent; background-clip: text; }

Google Fonts & Custom Fonts

HTML + CSS — Loading Custom Fonts
<!-- In <head> --> <link rel="preconnect" href="https://fonts.googleapis.com"> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet"> /* In CSS */ body { font-family: 'Inter', sans-serif; } /* Or use @font-face for self-hosted */ @font-face { font-family: 'MyFont'; src: url('fonts/myfont.woff2') format('woff2'); font-weight: 400 700; /* variable font range */ font-display: swap; }

Fluid Typography with clamp()

clamp(min, preferred, max) lets font sizes scale smoothly between a minimum and maximum value based on viewport width — no media queries needed.

CSS — Fluid Font Sizes
h1 { /* Minimum 1.8rem, scales with viewport, maximum 3rem */ font-size: clamp(1.8rem, 5vw, 3rem); } p { font-size: clamp(1rem, 2.5vw, 1.25rem); } /* Fluid line-height */ body { line-height: clamp(1.4, 1.2 + 0.5vw, 1.8); }
CSS Units for Typography

Use rem for font sizes (scales with the root element, respects user accessibility settings), em for spacing relative to the current font size, and px sparingly — only when you truly need a fixed size.

Wrapping Up

CSS gives you a rich color vocabulary — named colors, hex, RGB, HSL, and gradients — plus fine-grained control over every aspect of typography. Combining HSL colors, system font stacks, and fluid clamp() sizes produces accessible, responsive designs with very little code.
