CSS Grid Layout

CSS Grid is a two-dimensional layout system — it works with both rows and columns simultaneously. While Flexbox is great for one-directional flows, Grid excels at full page layouts, image galleries, and any design that needs to align content on both axes at once.

Enabling Grid

CSS — Activating Grid
.container { display: grid; }

Defining Columns and Rows

CSS — grid-template-columns & rows
/* Three equal columns */ .container { display: grid; grid-template-columns: 1fr 1fr 1fr; } /* Shorthand with repeat() */ .container { grid-template-columns: repeat(3, 1fr); } /* Mixed widths */ .container { grid-template-columns: 200px 1fr 2fr; } /* Define rows */ .container { grid-template-rows: 80px auto 60px; }
The fr Unit

fr stands for "fraction of available space." 1fr 2fr creates two columns where the second is twice as wide as the first. After fixed sizes (px, auto) are resolved, fr columns split the remaining space proportionally.

CSS — gap
.container { gap: 20px; /* row and column gap */ gap: 16px 24px; /* row-gap | column-gap */ }

Placing Items

CSS — grid-column & grid-row
/* Span across multiple columns or rows */ .featured { grid-column: 1 / 3; /* start at line 1, end at line 3 */ grid-column: span 2; /* shorthand: span 2 columns */ grid-row: span 2; /* span 2 rows */ } /* Place at a specific cell */ .logo { grid-column: 1; grid-row: 1; }

Named Grid Areas

Grid areas let you build page layouts with readable, visual template strings — no line numbers needed.

CSS — grid-template-areas
.layout { display: grid; grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; grid-template-columns: 200px 1fr 1fr; gap: 16px; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

auto-fit & auto-fill with minmax()

This is one of the most powerful Grid techniques — a responsive column layout with no media queries at all.

CSS — Responsive Grid Without Media Queries
.cards { display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); gap: 20px; } /* auto-fit: empty tracks collapse */ /* auto-fill: empty tracks are kept */ /* minmax(200px, 1fr): each column min 200px, max fills equally */

Grid vs Flexbox — When to Use Which

Use Flexbox when you have a single row or column of items and want them to share space naturally. Use Grid when you need to control both rows and columns simultaneously, or when you are building a macro page layout. They are complementary — use both in the same project.

Wrapping Up

CSS Grid — grid-template-columns, fr units, named areas, auto-fit with minmax() — gives you a declarative, readable two-dimensional layout system that is difficult to replicate with any other approach.
