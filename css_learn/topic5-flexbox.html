Flexbox Layout

Flexbox is a one-dimensional layout system — it works along either a row or a column. It is the perfect tool for navigation bars, card rows, centered content, and any time you need to distribute space between items or align them within a container.
Container vs. Items

Flexbox has two actors: the flex container (the parent you set display: flex on) and flex items (the direct children). Container properties control the axis, wrapping, and alignment of all items; item properties control individual sizing and ordering.

Enabling Flexbox

CSS — Activating Flex
.container { display: flex; /* or inline-flex for inline context */ }

gap

The gap property adds space between flex items without adding margin to the first or last item — cleaner than margins for most layouts.

CSS — gap
.container { display: flex; gap: 16px; /* row and column gap equal */ gap: 16px 24px; /* row-gap | column-gap */ row-gap: 16px; column-gap: 24px; }

CSS — Flex Item Sizing
/* Equal width columns that grow to fill space */ .col { flex: 1; } /* Fixed sidebar + growing main content */ .sidebar { flex: 0 0 260px; } /* don't grow, don't shrink, 260px */ .main { flex: 1; } /* take all remaining space */ /* Override alignment for one item */ .special { align-self: flex-end; } /* Visual reorder without changing HTML */ .logo { order: -1; } /* appears first */ .badge { order: 2; } /* appears last */

Perfect Centering with Flexbox

CSS — Centering Patterns
/* Horizontally and vertically centre a child */ .parent { display: flex; justify-content: center; align-items: center; min-height: 100vh; } /* Navigation bar — logo left, links right */ .navbar { display: flex; align-items: center; } .navbar .links { margin-left: auto; } /* push links to far right */
margin: auto in Flex

Setting margin-left: auto on a flex item consumes all available space to its left — a very powerful trick for pushing the item to the right end of the row.

Wrapping Up

Flexbox — display: flex, flex-direction, justify-content, align-items, gap, and the item properties — gives you a simple, readable way to build one-dimensional layouts. It is the best tool for navbars, card rows, and centring content.
